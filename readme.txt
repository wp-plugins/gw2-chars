Users can manage their own Guild Wars 2 characters. A list of all characters can be integrated in any post or page.

== Description ==

Every user who has the right to upload files can add his Guild Wars 2 characters with a screenshot and a short
phrase to the database.

Place the `[gw2chars]` shortcode on your page or article where you want to list all characters in the database.

== Installation ==

1. Upload all files to the `/wp-content/plugins/gw2-chars` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
